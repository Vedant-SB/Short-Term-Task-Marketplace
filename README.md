# Short-Term Task Marketplace

## Overview

Short-Term Task Marketplace is a platform that connects companies and individuals for short-duration tasks that can typically be completed within 3–7 days.

Companies can post short-term work requirements without going through a full hiring process, while students, freelancers, and professionals can discover tasks, apply, build experience, and earn money.

## Problem Statement

Many organizations have small tasks that do not justify hiring a full-time employee or intern.

At the same time, students and professionals are looking for practical work opportunities, portfolio projects, and short-term paid engagements.

This platform bridges that gap.
